# 系统使用教程
智能化工具作用是方便有效的管理每个HLOD System，如图。每个脚本都管理着一个系统，脚本引用着当前系统精细度模型、BVH、合批模型的跟节点。

2、一键更新

4、其他功能
4.2 一键生成、删除
一键生成、删除功能如图4-2所示，生成的节点名字后缀是一个递增的数字。

4.3 一键更新
一键更新功能如图4-3所示，当某个区域增加或删除了LOD Group时，使用更新功能只会更新有影响的部分，更新的包括网格和贴图以及保存在Asset下的资源。
